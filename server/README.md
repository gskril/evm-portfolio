# Server

Backend and API for the local-first EVM portfolio tracker.

## Endpoints:

### Accounts

- [ ] `GET /accounts`
- [ ] `GET /accounts/:address`
- [ ] `POST /accounts`

### Chains

- [ ] `GET /chains`
- [ ] `GET /chains/:id`
- [ ] `POST /chains`

### Tokens

- [ ] `GET /tokens`
- [ ] `GET /tokens/:address`
- [ ] `POST /tokens`
